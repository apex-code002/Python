# Snake Water Gun Game

This is a small Python project that I made while learning Python programming.

The game is based on Snake, Water, and Gun. The user plays against the computer, and the computer chooses a random option each time.

## How the Game Works

* Snake drinks Water → Snake wins
* Water sinks Gun → Water wins
* Gun kills Snake → Gun wins

If both the user and computer choose the same option, the game ends in a draw.

## Technologies Used

* Python 3
* Random Module

## How to Run

1. Download the project files.
2. Open the terminal in the project folder.
3. Run:

python main.py

4. Enter:

   * s for Snake
   * w for Water
   * g for Gun
